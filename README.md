# Kibble 🐕

용어 검색 도구 — 번역 메모리(TM) 및 용어집(TB) 파일에서 용어를 빠르게 검색합니다.

## 기능

- **파일 지원**: xlsx, csv, txt 파일 로드 (드래그 앤 드롭 또는 Ctrl+O)
- **검색 방식**: 부분 일치(Substring) + 유사 일치(Fuzzy) 검색
- **원문/번역문 검색**: 원문(한국어) 또는 번역문(대상 언어)에서 검색 선택 가능
- **번역문 필터**: 검색 결과를 번역문 텍스트로 추가 필터링
- **다국어 지원**: 한국어 원문 → 영어(메인), 일본어, 중국어 등 다국어 대상
- **열 자동 감지**: 언어 코드 헤더 인식 또는 한글 콘텐츠 기반 자동 열 매핑
- **열 수동 설정**: ⚙ 버튼으로 원문/번역문 열을 직접 지정 가능
- **검색 결과 하이라이트**: 검색어는 노란색, 필터 텍스트는 파란색으로 강조 표시
- **대응 번역 강조**: 검색된 원문의 대응 번역문에 연한 배경색 적용
- **보기 모드**: 3열 보기(원문 + 선택 언어 + 메타) 또는 원문 + 전체 번역문
- **일치율 표시**: 검색 결과에 일치율(%) 열 표시
- **열 정렬**: 열 헤더 클릭으로 알파벳/수치 정렬
- **줄 바꿈**: 긴 텍스트를 셀 내에서 줄 바꿈하여 표시
- **검색 애니메이션**: 검색 중 강아지 공 가져오기 애니메이션 + 경과 시간 표시
- **세션 저장**: 마지막 파일, 검색 설정, 창 크기를 자동 저장/복원
- **EN/KO UI**: 영어/한국어 인터페이스 전환 (좌측 하단)
- **전역 단축키**: `Ctrl+Shift+K`로 Excel, LibreOffice 등 외부 앱에서 선택한 텍스트를 바로 검색
- **포터블**: 설치 불필요, 단일 exe 파일

## 사용법

### 파일 로드
1. 파일을 창에 드래그 앤 드롭하거나 `Ctrl+O`로 파일 탐색기를 엽니다.
2. xlsx, csv, txt 파일을 지원합니다.
3. 열이 자동으로 감지됩니다. 잘못된 경우 ⚙ 버튼으로 수정하세요.

### 검색
1. "검색어" 입력란에 검색할 용어를 입력합니다.
2. `Enter` 키를 누르거나 "검색" 버튼을 클릭합니다.
3. 검색 방향을 선택합니다:
   - **원문에서 검색**: 한국어 원문 열에서 검색
   - **번역문에서 검색**: 대상 언어 열에서 검색 (3열 보기에서 선택된 언어)
4. "번역문 필터"로 결과를 추가로 필터링할 수 있습니다 (`Enter`로 적용).

### 검색 옵션
- **일치 방식**: 부분 일치 / 유사 일치 / 모두
- **대소문자 구분**: 영문 대소문자 구분 여부
- **와일드카드 추가**: `*`를 와일드카드로 사용 (부분 일치 모드에서만)
- **최소 일치율**: 슬라이더로 최소 일치율 조절 (기본: 50%)
- **최대 결과**: 표시할 최대 결과 수 (기본: 200)

### 단축키

| 단축키 | 기능 |
|--------|------|
| `Ctrl+F` | 검색창 포커스 |
| `Ctrl+O` | 파일 열기 |
| `Ctrl+C` | 선택한 셀 복사 |
| `Ctrl+W` | 현재 파일 탭 닫기 |
| `Escape` | 검색창 초기화 |
| `Ctrl+Shift+K` | 전역 검색 (외부 앱에서 텍스트 선택 후 사용) |

### 결과 조작
- **셀 클릭**: 셀 선택
- **더블 클릭**: 셀 텍스트를 클립보드에 복사
- **Ctrl+C**: 선택한 셀 복사
- **우클릭**: 행 복사, 셀 복사, 원문만 복사, 번역문만 복사
- **열 헤더 클릭**: 해당 열로 정렬 (오름차순/내림차순)
- **열 경계 드래그**: 열 너비 조절
- **행 경계 드래그**: 행 높이 조절

### 외부 앱 연동 (전역 단축키)

Kibble이 실행 중일 때, Excel, LibreOffice, 메모장 등 **어떤 앱에서든** 용어를 검색할 수 있습니다:

1. 외부 앱에서 검색할 텍스트를 **`Ctrl+C`로 복사**합니다.
2. `Ctrl+Shift+K`를 누릅니다.
3. Kibble이 자동으로:
   - 클립보드의 텍스트를 검색창에 입력
   - 검색 실행
   - 창을 앞으로 가져옴

> Kibble이 백그라운드에 있어도 (최소화 상태에서도) 작동합니다.

## 기술 스택

| 구성 요소 | 기술 |
|-----------|------|
| 런타임 | Python 3.11+ |
| GUI | PyQt6 |
| 파일 파싱 | pandas + openpyxl |
| 유사 검색 | RapidFuzz |
| 인코딩 감지 | chardet |
| 패키징 | PyInstaller |

## 빌드

### 환경 설정
```bash
cd kibble
python -m venv .venv
.venv/Scripts/pip install -r requirements.txt
```

### 테스트 실행
```bash
.venv/Scripts/python -m pytest tests/ -v
```

### 실행 (개발 모드)
```bash
.venv/Scripts/python main.py
```

### exe 빌드
```bash
.venv/Scripts/pip install pyinstaller
.venv/Scripts/pyinstaller Kibble.spec
```

빌드 결과: `dist/Kibble.exe` (약 70MB, 포터블)

## 파일 구조

```
kibble/
├── main.py                 # 진입점
├── core/
│   ├── parser.py           # 파일 파싱 (xlsx, csv, txt)
│   ├── search.py           # 검색 엔진 (부분 일치 + 유사 일치)
│   └── session.py          # 세션 저장/복원
├── ui/
│   ├── main_window.py      # 메인 윈도우
│   ├── search_panel.py     # 검색 패널
│   ├── results_table.py    # 결과 테이블 + 하이라이트
│   ├── file_tabs.py        # 파일 탭
│   ├── drop_zone.py        # 드래그 앤 드롭 + 토스트 알림
│   ├── column_mapper.py    # 열 설정 다이얼로그
│   └── i18n.py             # EN/KO 번역
├── workers/
│   ├── parse_worker.py     # 파일 파싱 백그라운드 워커
│   └── search_worker.py    # 검색 백그라운드 워커
├── assets/
│   └── icon.ico            # 앱 아이콘
├── tests/                  # 테스트 (42개)
├── Kibble.spec             # PyInstaller 빌드 설정
└── requirements.txt        # 의존성
```

## 제한 사항

- 최대 총 항목 수: 500,000 (파일 합산)
- 한국어 조사 처리 미지원 (v1): "번역을", "번역이" 등은 부분 일치로 검색 가능
- xlsx 파일은 첫 번째 시트만 로드
- 검색어 최소 길이: 2자
