# 변경 이력

## v2.0 — 2026-03-18

### 새로운 기능
- **웹 버전 추가** — 브라우저에서 바로 사용할 수 있는 Kibble Web (https://hkkim-wm.github.io/Kibble/). 데스크톱과 동일한 기능 제공 (전역 단축키 제외).
  - React 18 + TypeScript + Tailwind CSS 기반
  - 가상 스크롤 (@tanstack/react-virtual)
  - Web Worker 백그라운드 검색
  - GitHub Pages 호스팅
  - 68개 단위 테스트
- **다크 모드** — 설정 메뉴에서 전환 가능. 검색 하이라이트 색상도 다크 모드에 맞게 자동 조정. (데스크톱 + 웹)
- **단어 단위 검색** — "단어 단위" 체크박스 추가. 영문 단어 경계 기반 매칭, CJK 텍스트는 부분 문자열로 자동 fallback. (데스크톱 + 웹)
- **글꼴 크기 조절** — 검색 결과 테이블의 글꼴 크기를 S/M/L/XL로 조절 가능. (데스크톱 + 웹)
- **디렉토리 일괄 로드** — 파일 > 폴더 열기로 폴더 내 모든 xlsx/csv/txt를 한번에 로드. (데스크톱 + 웹)
- **레이아웃 분할 조절** (웹) — 검색 패널과 결과 테이블 사이의 경계를 드래그하여 비율 조절 가능.

### 개선 사항
- **Settings 메뉴** — 기존 Help 메뉴를 Settings로 변경. 언어 전환, 다크 모드, About을 하나의 메뉴에서 관리. 하단 언어 전환 버튼 제거. (데스크톱 + 웹)
- **언어 헤더 확장** — 새로운 컬럼 헤더 인식 추가:
  - KO: `Source`
  - CT: `zh-Hant`, `TW`
  - CS: `zh-Hans`
  - ES: `ES-LATAM`, `Spanish (Latin America)`
- **파일 간 열 병합 개선** — 열 이름을 정규화하여 다른 파일의 "English"와 "EN"을 동일 열로 통합
- **열 너비 조절** (웹) — 헤더 경계 드래그로 열 너비 조절 가능
- **상태바 간소화** — 검색 완료 시간만 간결하게 표시. 불필요한 요소 제거.
- **전역 단축키 설명 개선** (데스크톱) — 단축키의 기능을 명확하게 안내

### 버그 수정
- **퍼지 스코어 수정** (웹) — token_set_ratio 제거. 부분 일치 시 100%로 부풀려지던 문제 해결.
- **탭 아래 빈 공간 제거** (데스크톱) — 전체 검색 탭 아래에 불필요하게 표시되던 빈 영역 제거.

## v1.1 — 2026-03-17

### 새로운 기능
- **메뉴 바** — 파일 메뉴(파일 열기, 종료)와 도움말 메뉴(Kibble 정보)를 추가.

### 개선 사항
- **파일 간 열 병합** — 여러 파일을 동시 검색할 때 동일 언어를 가리키는 열을 자동 병합. 17개 언어 코드 지원.
- **중국어 번체/간체 분리** — CT (zh-TW) 및 CS (zh-CN)를 별도 타겟으로 인식.
- **열 정렬 순서** — 번역 열 → 참고 정보 열 → 메타데이터 열 순으로 자동 정렬.
- **불일치 감지 개선** — 번역 열만 비교하고 메타데이터/참고 열은 무시.

### 버그 수정
- **혼합 파일 검색 시 충돌 수정** — xlsx와 csv 파일을 함께 검색할 때 발생하던 충돌 해결.
- **열 순서 유지** — 검색 결과에서 원본 파일의 열 순서를 유지.
- **검색 워커 경합 조건 수정** — 새 검색 시작 전 이전 검색 워커의 시그널 해제.
